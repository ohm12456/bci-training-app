Design a modern medical Brain-Computer Interface (BCI) training application for hospitals.

Style: clean medical interface, minimal, professional, calm colors.

Primary colors: white, soft blue, teal.
Accent colors: green for success, red for error, yellow for warning.

Target user: neurologists and doctors using EEG-based motor imagery rehabilitation.

The interface should feel similar to hospital software: structured, readable, and efficient.

Create the following screens:

Login Screen
Doctor login interface with hospital branding.
Fields: email/doctor ID, password, login button.
Clean medical layout with a soft gradient background.

Dashboard
Overview panel showing:

EEG device connection status (simulated)

signal quality indicator

quick patient statistics

latest training sessions

large “Start Training Session” button

Patient Management
Patient list table with:

patient name

age

medical condition

last session accuracy

buttons: add patient, edit patient, delete patient

Patient Detail Page
Display patient profile information:

name

age

condition

doctor notes

Include:

training history table

progress chart showing improvement over time

“Start Training” button

Training Session Interface
Main BCI training screen.

Sections:

patient info panel

EEG signal visualization (simulated waveform graph)

task instruction area

Training flow UI:
Step 1: instruction appears (LEFT HAND or RIGHT HAND imagery)
Step 2: countdown timer (3-2-1)
Step 3: signal acquisition phase (4 seconds)
Step 4: prediction result display

Show:
Expected task vs Model prediction

Display success or failure clearly with color indicators.

Session Summary Screen
After training show:

total trials

correct predictions

accuracy percentage

simple performance chart

Doctor Account Page
Doctor profile information:

name

hospital

department

email

settings to edit profile
